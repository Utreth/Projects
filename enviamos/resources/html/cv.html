Proyectos

Desarrollo de un sistema de gestión de envíos y almacenamiento para la Transportadora Chillys

Como parte de mi proyecto académico, se diseño e implemento un sistema para la empresa de logística Transportadora Andina, que permite gestionar el almacenamiento y los envíos nacionales. El sistema permite a los clientes almacenar mercancías en bodegas o enviar diferentes tipos de paquetes, sobres, cajas y bultos, con un cálculo de costos según peso, volumen, fragilidad y valor declarado. Además de gestionar el seguimiento de los envíos, que incluye su estado (preparación, enviado, fallos en entrega, etc.), el sistema también ofrece la opción de obtener un informe detallado en formato JSON para cada envío. Durante el desarrollo, abordé desafíos como el diseño flexible y extensible del sistema para agregar nuevos tipos de envíos sin modificar el código existente. Implementé validaciones de datos y procedimientos de generación automática de identificaciones, así como el cálculo preciso de costos basados en las características de los envíos. También utilicé diagramas UML para modelar las clases del sistema y garantizar la correcta implementación de la lógica de negocio. Este proyecto me permitió integrar mis conocimientos de backend y front-end, y me resultó una experiencia muy interesante y didáctica.
